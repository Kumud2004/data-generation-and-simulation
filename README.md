# data-generation-and-simulation
Simulation Tool Used
NumPy and SciPy were used to simulate a non-linear system with random parameters.

Methodology
Random values of system parameters were generated within defined bounds and passed to a simulated mathematical model. A total of 1000 simulations were performed to generate synthetic data.

Machine Learning Models
The generated data was used to train and evaluate the following models:

Linear Regression
Decision Tree
Random Forest
Support Vector Regression
K-Nearest Neighbors
Evaluation Metrics
Models were evaluated using Mean Squared Error (MSE) and R² Score.

Result
Random Forest Regressor achieved the best performance with the highest R² score and lowest error.
